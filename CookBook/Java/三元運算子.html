# 三元運算子

可以看成一個簡化的 if-else

# 運算子

運算子是一種符號或關鍵字，用於執行特定的數學或邏輯操作。

ChatGPT解釋： 常見的運算子包括算術運算子（如加號和減號）、比較運算子（如等於和不等於）、邏輯運算子（如AND和OR）以及位元運算子（如位元左移和位元右移）。運算子可以用來操作變量、常量和表達式，從而實現數學運算、比較、邏輯操作等。在不同的程式語言中，運算子的表示方式和功能也會有所不同。熟練掌握運算子是程式設計的基礎之一。

因為要三個值才可進行運算，可以叫三元運算子 Java的三元運算子：

條件 ? 為真時 : 為假時 ;

例子

a = a < 1 ? a : 1;

# 好處

  • 壓縮代碼
    • 以前電腦儲存空間小，就是靠 三元運算 寫少 幾行 if-else。
  • 提高運行速度
    • 寫小量code可能不明顯，但寫一個大型程式時，速度會明顯加快
  • 增加可讀性
    • 可能新手會看不懂，不如要知道，學就看得明。加油
    • 越小代碼，越容易 維護，越容易 看得完代碼，越容易 理解（都要寫好注解哦）。

本人非常喜歡 三元運算, 請多多使用~
