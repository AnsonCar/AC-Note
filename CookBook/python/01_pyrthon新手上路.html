# 01_python新手上路

學progamm是一條不歸路， 學會以後，觀察世界的方式，都會不一樣。

# Python介紹

python主要用於 數據分析、人工智能與機器學習、自動化、爬蟲、Wed框架等，因為語法簡單容易上手，適合新手入門學習。

# 環境準備

# 安裝 python 及 相關IDE

IDEopen in new window: Jupyter Lab, Visual Studio Code

# 安裝 python

  1. 打開 https://www.python.org/
  2. 按 Downloads 跳到 Downloads頁面
  3. 按 Download Python 3.XX.X
    註: 默認安裝python 最新版本，我的 Python版本為 3.11.0，如果想要其他版本可以下拉找出自己想要的版本

# 安裝 Visual Studio Code

  1. 前往 https://code.visualstudio.com/
  2. 按 Download XXX
  3. 安裝軟件
  4. 順利打開會提示你安裝 中文套件（看你的需求）
  5. 安裝延伸模組 Python
    註: VSc是個非常好用的IDE, 其他語言都會用到(e.g JavaScript, Java)

# 安裝 Jupyter Lab

  1. 打開 終端機 (Terminal)
    • Mac : 在啟動台 打開 終端機
    • Win : 搜尋 cmd
  2. 輪入 pip install jupyterlab
    • 等待完成，不要急，電腦會下載很多文件，不要中途關閉 Terminal。
  3. 下載完成後，在 Terminal 輸入 jupyter-lab
    • 之後要打開 Jupyter Lab 都是這樣做
    • 副檔名 .inpynb

# 線上環境

如果不想下載上述內容，或想使用平板手機去寫python的話，可以使用以下線上環境。 菜鳥python在线编辑open in new window

# 查看 Python 版本

# 在 終端機
python3 -V  
python3 --version
# 在 Jupyter Lab
# 在 python 程式塊 加 ! 可以使用 終端命令
!python3 -V
!python3 --version
